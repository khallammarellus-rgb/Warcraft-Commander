Warcraft Commander

Build an interactive Azeroth globe in Google Earth Pro and run pass-and-play wargame campaigns — markers, turns, blind play, and hosted mode.

Get started

  1. Clone or download this repo for build scripts and config.
  2. Export minimap tiles with wow.export.
  3. Run python3 scripts/build_world_globe.py --variant wowcommanderalpha.
  4. Open 03-kml/wowcommanderalpha/doc.kml in Google Earth Pro.
  5. Launch scripts/WOW Commander.command to set up a campaign.
View on GitHub

Campaign play

  • Install the globe once — turns ship markers only as small KMZ files.
  • Setup wizard: factions, HQs, blind mode, cell assignment.
  • Export turns with package_wargame_client.py for Discord or email pass-and-play.

Azeroth Explorer (maps only)

Want just the map — no wargame layer? See Azeroth Explorer, a separate standalone download.
