Getting started

New player — 3 steps (like Azeroth Explorer)

1. Install Google Earth Pro

Download Google Earth Pro (free).

2. Download the map

  1. Open github.com/khallammarellus-rgb/Warcraft-Commander
  2. Click green Code → Download ZIP
  3. Unzip the folder on your computer

Do not use old GitHub Releases (player-v3 split zip) or the automatic Source code asset on release pages — use Code → Download ZIP on the main repo.

3. Open the map

  1. In Google Earth Pro: File → Open
  2. Select player/WoW Commander.kml inside the unzipped folder
  3. Tick WOW Commander Alpha (play) in Places

Keep player/kml/, player/tiles/, and WoW Commander.kml together.

4. Your table (hosted campaign)

Campaign markers refresh from this portal (~60s). Right-click Campaign Board links → Refresh after turns update.

Upload tokens are issued by the organizer (white cell).

5. Optional — Python player menu

Only for local setup, sync, or turn export. Not required for hosted portal play.

cd player
python3 -m pip install -r requirements.txt

Mac: double-click player/scripts/WOW Commander.command

6. Custom unit icons (optional)

Use the Unit Icon Builder to compose military unit markers (shape + identifier + WoW class overlay).

Download PNGs into player/assets/player_custom_icons/ and reference them on placemarks — see README in that folder.

7. Turn loop

  • Edit markers in Google Earth Pro → save
  • Upload your KMZ on your table page when it is your cell's turn
  • Right-click Campaign Board links → Refresh after white cell merges
